## ✈️ About

**JK Travel** is an enterprise billing and management system built for a travel agency in Panama. Developed with Flutter for a cross-platform desktop and mobile experience, backed by SQL Server for reliable data management.

The system replaced entirely manual and paper-based workflows, centralizing client management, billing, and financial reporting into one unified platform.

> 🏢 Private client system — live in production.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🧾 **Invoice Generation** | Create and manage professional invoices for travel packages |
| 👥 **Client Management** | Full client registry with history and contact info |
| 📦 **Package Billing** | Itemized billing for flights, hotels, tours, and custom packages |
| 📊 **Financial Reports** | Sales summaries, revenue tracking, and exportable reports |
| 🔐 **Role-Based Access** | Secure login with user roles and permissions |

## ✈️ Descripción

**JK Travel** es un sistema empresarial de facturación y gestión desarrollado para una agencia de viajes en Panamá. Construido con Flutter para una experiencia multiplataforma (desktop y móvil), respaldado por SQL Server para una gestión de datos confiable.

El sistema reemplazó completamente los flujos manuales y en papel, centralizando la gestión de clientes, facturación y reportes financieros en una sola plataforma.

> 🏢 Sistema privado de cliente — en producción.

---

## ✨ Características

| Característica | Descripción |
|---|---|
| 🧾 **Generación de Facturas** | Crear y gestionar facturas profesionales para paquetes turísticos |
| 👥 **Gestión de Clientes** | Registro completo de clientes con historial e información de contacto |
| 📦 **Facturación de Paquetes** | Desglose de vuelos, hoteles, tours y paquetes personalizados |
| 📊 **Reportes Financieros** | Resúmenes de ventas, seguimiento de ingresos y exportación de reportes |
| 🔐 **Acceso por Roles** | Login seguro con roles y permisos de usuario |
