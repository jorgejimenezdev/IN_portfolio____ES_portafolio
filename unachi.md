## 📬 About

Institutional web platform developed for **Universidad Autónoma de Chiriquí (UNACHI)**. The system gives students a formal channel to submit complaints, suggestions, and feedback — while giving administrators a structured workflow to manage, respond to, and resolve each case.

Built with SQL Server for data reliability and CodeIgniter 4 for a clean MVC architecture. OWASP Top 10 principles applied throughout to protect sensitive student data.

> 🎓 University-scale deployment — handling real institutional data.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📝 **Complaint Submission** | Students submit complaints with category, description, and files |
| 🔄 **Lifecycle Tracking** | Full status tracking: submission → review → resolution |
| 👥 **Role-Based Access** | Separate interfaces for students and administrators |
| 📊 **Admin Dashboard** | Overview of all cases with filters and status management |
| 🔐 **Secure by Design** | OWASP-aligned: input validation, XSS prevention, auth controls |

## 🔐 Security Practices Applied

| Area | Implementation |
|---|---|
| **Injection** | Prepared statements and query builder — no raw SQL |
| **XSS** | Output escaping on all user-generated content |
| **Authentication** | Session-based auth with role enforcement |
| **Access Control** | Route-level middleware for role verification |
| **Input Validation** | Server-side validation on all forms |

## 📬 Descripción

Plataforma web institucional para la **Universidad Autónoma de Chiriquí (UNACHI)**. El sistema da a los estudiantes un canal formal para enviar quejas, sugerencias y comentarios — con un flujo estructurado para que los administradores gestionen y resuelvan cada caso.

Construido con SQL Server para confiabilidad de datos y CodeIgniter 4 para una arquitectura MVC limpia. Principios OWASP Top 10 aplicados para proteger datos estudiantiles sensibles.

> 🎓 Despliegue a escala universitaria — manejando datos institucionales reales.

| 👥 **Acceso por Roles** | Interfaces separadas para estudiantes y administradores |
